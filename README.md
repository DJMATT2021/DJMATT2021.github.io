# E‑Bike Web Bluetooth Dashboard

This project is a simple Web Bluetooth dashboard for my e‑bike.  
It runs directly in Safari on iPhone and connects to a Bluetooth device that sends data such as speed.

## 🚴‍♂️ Features
- Bluetooth device selector
- Demo speed gauge
- Works on iPhone, iPad, and desktop browsers that support Web Bluetooth
- Designed to be added to the Home Screen for quick access

## 📁 Files
- **index.html** — main dashboard page  
- (Optional) **script.js** — extra JavaScript if needed  
- (Optional) **style.css** — custom styling
